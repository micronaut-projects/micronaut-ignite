2.2 Ignite Thin Client Configuration

Version: 1.0.0.BUILD-SNAPSHOT

2.2 Ignite Thin Client Configuration

Ignite supports a thin-client configuration that does not become apart of the topology. A minimum thin client configuration is provided below:

Configuring Ignite Thin Client
ignite:
  enabled: true
  thin-clients:
    default:
      address:
      - "127.0.0.1:10800"
      - "127.0.0.1:10801"
