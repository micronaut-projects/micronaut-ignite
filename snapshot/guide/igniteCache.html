3 Micronaut Cache

To take advantage of Micronaut Cache, this can be enabled by including the following dependency:

implementation("io.micronaut.ignite:micronaut-cache-ignite")
<dependency>
    <groupId>io.micronaut.ignite</groupId>
    <artifactId>micronaut-cache-ignite</artifactId>
</dependency>

Note: Information usage in regards to micronaut cache can be found in the micronaut cache documentation.

3.1 Ignite Client Cache

Configuration Ignite Cache
ignite:
  enabled: true
  clients:
    default:
      path: "classpath:default.cfg"
  thin-caches:
    default:
      client: "default"
